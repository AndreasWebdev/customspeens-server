Usage Policy

Our Usage Policy serves as the Terms of Service of the SpinShare API. Please follow this ruleset if you are using the SpinShare API.

DO

  • Protect userdata.
  • Provide users with information about which data you access
  • Add a label or a backlink to SpinShare

DON'T

  • Use API data for a purpose not connected to SpinShare or Spin Rhythm XD
  • Obtain passwords or other credentials
  • Use userdata for marketing or advertisements
  • Sell or license API data
  • Scrape API data (caching is excluded from this)
  • Use this API data to do anything illegal or harmful

We currently don't ratelimit our open APIs. However, we still would like you to keep an eye on your API usage and cache API data whenever possible.
